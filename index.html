Password Generator

This is my web based random password generator, a project I built using HTML, CSS and JavaScript for the page layout, styling and functionality respectivley. The "range and "checkpoint" inputs on the main HTML file allow the user to choose the password length, and the inclusion of upper cae letters, numbers and symbols, using a slider and tickboxes on the UI. The functions in the JavaScript end of the app randomly chooses characters using the charCode method, which assigns each ascii to a number ranging from 0 to 255. It then appends them to an initially empty list, and returns the result as a string displayed in the password display. the getElementsById and AddEventListener methods take in information inputted by the user so that the JS code can process it.

Calculator

This is a web based caluculator. The layout was designed using HTML and CSS, with CSS especially being use to arrange each of the number and operand buttons as a grid below the display. In the JS code the calculator is written as it's own class, with functions for appending and removing numbers as a string on the display, choosing operands, and a "compute" function to make the actual calculations and return the result. There is also a function to update the display each time an input is made by the user.
